# apps/frontend/src/features — 에이전트 라우팅

`apps/frontend/src/features/**` 작업에 적용된다. 더 가까운 `AGENTS.md`가 있으면 그 파일이 우선한다.

기능별 수직 슬라이스 폴더다.
`auth/`는 GitHub OAuth 로그인 기능의 owner 전속 경로다(owner는 루트 AGENTS.md §3 참조).
owner가 아니면 직접 수정하지 않고 Issue·PR 코멘트로 제안한다.
`landing/`은 랜딩 페이지 기능 슬라이스다.
폴더 구조 규약은 `README.md`가 원본이다.
