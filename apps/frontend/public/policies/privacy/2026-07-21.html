개인정보 수집·이용

정책 버전: 2026-07-21

수집 항목과 목적

GitHub 계정 식별자·로그인명·이름·프로필 이미지, 사용자가 입력한 프로필·연락 정보, 역할과 동의 기록을 계정 식별·접근 제어·프로그램 운영에 사용합니다.

보유와 삭제

회원정보와 개인 연결값은 회원 자격과 서비스 운영에 필요한 동안 보유합니다. 탈퇴나 연결 해제 시 개인 프로필과 개인 단위 공개 연결은 삭제 또는 비식별화하고 이후 노출을 중단합니다.

Org 자산과의 분리

개인정보 처리는 Org 저장소, Git 이력, 저장소 식별자와 저장소 단위 누적 지표를 자동으로 삭제하지 않습니다. 예외적인 정정·삭제는 플랫폼 운영 절차로 처리합니다.
