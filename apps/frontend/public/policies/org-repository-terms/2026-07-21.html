Org 저장소 운영 약관

정책 버전: 2026-07-21

소유·관리와 참여 권한

JNU-SWCU Org 안에 생성·이관된 저장소의 소유·관리 주체는 Organization입니다. 사업단 staff는 프로그램 운영을 위한 관리·접근·가시성 변경 권한을 가지며, 학생과 팀은 자신에게 배정된 저장소의 기여·사용·접근 권한을 가집니다.

가시성과 공개 전환

private 저장소 접근은 Platform ADMIN/STAFF와 해당 팀 구성원으로 제한합니다. 공개 전환은 정해진 마일스톤 이후 권한 있는 운영자의 명시적 동작으로 수행하며 자동 공개하지 않습니다.

보존과 예외 처리

회원 탈퇴·연결 해제, 행사 종료와 가시성 변경은 Org 저장소, Git 이력, 저장소 식별자와 누적 지표를 삭제하거나 초기화하지 않습니다. 법적 삭제 명령, 권리 침해 신고와 잘못 수집된 정보의 정정은 예외적인 운영 절차로 처리합니다.

코드 저작권과 IP

이 약관에 동의하는 것만으로 코드 저작권이나 IP가 사업단에 이전되거나 공동 저작권이 성립하지 않습니다. 라이선스와 성과물 IP 조건이 필요한 프로그램은 별도 약관으로 고지합니다.
