GitHub 활동 수집·공개 범위

정책 버전: 2026-07-21

수집 범위와 목적

JNU-SWCU Org 저장소의 식별자·가시성, 활동 시각·종류와 저장소 단위 최소 집계값을 프로그램 운영과 커뮤니티 활성화 지표 산출에 사용합니다. 개인 계정 소유 저장소는 이 정책의 수집 대상이 아닙니다.

수집하지 않는 정보

raw payload, 소스 코드, 본문, diff, commit message와 author email은 저장하지 않습니다.

공개 범위

private 저장소는 Platform ADMIN/STAFF와 해당 팀 구성원으로 접근을 제한합니다. 저장소와 활동 정보를 자동 공개하지 않으며, 공개 전환은 정해진 시점 이후 권한 있는 운영자의 명시적 동작으로만 수행합니다.
