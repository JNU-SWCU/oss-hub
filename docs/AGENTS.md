# docs — 에이전트 라우팅

`docs/**` 작업에 적용된다. 더 가까운 `AGENTS.md`가 있으면 그 파일이 우선한다.

`rules/`는 커밋·PR·보안·frontend·CI 검증 규칙의 원본이다.
`decisions/`는 ADR이며 결정이 바뀌면 새 파일이 아니라 같은 문서를 갱신한다.
`exec-plan/active/`는 진행 중 기능의 실행 계획, `exec-plan/archive/`는 완료된 계획이다.
`handoff/`는 TEAM-STATE 스냅샷과 GitHub drift 검사 문서다.
이 트리의 모든 문서는 한 문장을 한 줄에 쓴다 — 루트 AGENTS.md의 문서 스타일 규칙을 따른다.
